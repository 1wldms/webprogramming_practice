What is Style-It?

Style-It is a weather-based outfit recommendation web application developed by Seoyun, Jieun, and Naeon as part of the Web Programming course at Yonsei University. This web application recommends outfit combinations based on the current weather.
We aim to help you choose suitable clothing for unfamiliar climates or seasonal conditions!
